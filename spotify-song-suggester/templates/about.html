Methodology

We started with a dataset containing over 170k songs from The Spotify API. Then, using features such as 'danceability', 'key', 'popularity', 'tempo', 'valence', and 'year', we use a MinMaxScaler from scikit-learn to scale all features between 0 and 1. Next, we use Principal Component Analysis (PCA) for Dimensionality Reduction to reduce the number of features from 15 to 7. Finally, we use a Nearest Neighbors model (using Manhattan distance) to find and return the 10 most similar songs to the user's input.
